# AI 漫导 (DirectorAI) 用户手册

欢迎使用 **AI 漫导**！这款 App 能帮您把一个简单的创意变成一段精彩的视频。

## 1. 快速开始

### 第一步：设置 API Key
首次打开应用，请点击右上角的 **设置 (Settings)** 图标，进入配置页面。
您需要输入以下服务的 Key 才能正常使用：
*   **智谱 AI**: 用于生成剧本。
*   **Tuzi (Image/Video)**: 用于生成画面和视频。

### 第二步：开始创作
1.  回到主页（聊天界面）。
2.  在底部输入框告诉 AI 您的想法，例如：
    > "帮我做一个关于小猫在太空探险的视频，温馨风格"
3.  **AI 导演** (GLM-4.7) 会分析您的需求，并为您规划一个包含多个分镜的剧本。

## 2. 功能详解

### 2.1 剧本预览 (Script Preview)
当剧本生成完毕，您会看到一个卡片，展示了：
*   **剧本标题**
*   **场景列表**：每个场景的旁白和画面描述。
*   **视觉预览**：角色一致性分析结果。

点击 **"确认生成"**，AI 将开始为您绘制画面。

### 2.2 视频生成流程
这是一个自动化的过程，您可以在聊天界面看到实时进度：
1.  **绘制分镜 (Drawing)**: AI 为每个场景生成静态图片。
2.  **生成视频 (Filming)**: 将静态图片转化为 5秒 的动态视频。
3.  **合成 (Assembly)**: (仅安卓) 自动将所有片段合并为一个长视频。

### 2.3 画廊 (Gallery)
所有生成的视频都会自动保存到您的手机相册 (`Movies/DirectorAI` 目录)，您也可以在 App 内的画廊页面回顾历史作品。

## 3. 常见问题 (FAQ)

**Q: 生成过程需要多久？**
A: 取决于场景数量。通常每个场景需要 1-2 分钟。7个场景的剧本大约需要 10-15 分钟。您可以将其挂在后台运行。

**Q: 为什么生成的视频里人物长得不一样？**
A: AI 目前在保持人物一致性方面仍有挑战。我们使用了"角色卡"技术来尽量维持特征，但无法保证 100% 完美。尝试在描述中更详细地指定人物特征（如"蓝头发的女孩"）会有帮助。
